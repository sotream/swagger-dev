## Project technologies:

1. [Express](https://expressjs.com)
2. [PostgreSQL](https://www.postgresql.org)
3. [Swagger](https://swagger.io)

### 1. Install dependencies

```bash
npm install
```

### 2. Create local `.env` file

```yaml
POSTGRES_DATABASE="***"
POSTGRES_HOST="***"
POSTGRES_PASSWORD="***"
POSTGRES_USER="***"
```

### 3. Running the app in dev mode

```bash
npm run dev
```
